#  Credit Limit Calculator
This is a simple credit limit calculator coded with c++. This program calculates and display credit's result based on user's balance and credit limit. It uses functions, arrays, input validation and formatted output. It includes colors and effects that could enhance user experience.

## Tech Stack
- [C++](https://isocpp.org)

## Developing
Compile and run the main.cpp file
```bash
clang++ main.cpp -o main && ./main
```
